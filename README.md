## About Me

I'm a PM who came up through API integration engineering — MuleSoft, Apigee, building the plumbing before I learned to own the product. That path shapes everything: I think in systems, I prototype before I spec, and I'm drawn to problems that don't have clean solutions yet.

- 🏢 **Now:** Senior PM at **Avasant** — built a S2P + CLM platform from zero to **$10M+ ARR**. Avasant MVP Award 2024.
- ⚙️ **Origin:** IT Analyst @ TCS (MuleSoft · Apigee · IBM API Connect) → MBA NMIMS Mumbai (Dean's List) → PM
- 🤖 **Obsession:** What happens when AI meets messy, high-stakes enterprise workflows
- 🔍 **Philosophy:** GitHub is where I make my reasoning visible — every repo starts with a user problem I've actually felt
- 📍 **Looking for:** Senior → Staff PM · Platform · AI · Enterprise SaaS · Bangalore · Remote-friendly

| **LeapFrogX** *(private)* | Job seekers prep in spreadsheets with zero AI coaching | AI career platform — mock interviews with scoring, ATS resume analysis, per-role prep journeys | React · GPT-4o · MongoDB |

| **SimplySocial** *(private)* | Managing multiple social accounts requires too many separate apps | Unified social dashboard — connect platforms, compose posts, cross-platform analytics | React · Node · MongoDB |
